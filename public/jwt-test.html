How to Get a Real Token

To test with a real token from your OIDC provider:

  1. Authenticate with your OIDC provider
  2. Extract the access token or ID token from the authorization response
  3. Paste the token in the field above
  4. Configure OIDC environment variables for verification:
    • OIDC_JWKS_URI
    • OIDC_ISSUER
    • OIDC_AUDIENCE
